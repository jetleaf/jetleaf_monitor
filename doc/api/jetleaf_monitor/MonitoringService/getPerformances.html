List<Performance> getPerformances()

Returns a list of all recorded Performance snapshots currently maintained by the monitoring subsystem.

Each Performance entry represents aggregated execution metrics for a specific monitored method or class, typically created via:

  • @Monitor() annotations
  • Explicit performance tracking calls within the application

Example

final performances = monitoring.getPerformances();
for (final perf in performances) {
  print('${perf.getName()}: ${perf.getRunPeriod()}');
}

Use Cases

  • Dashboard rendering of all performance metrics
  • Bulk export for telemetry or diagnostics
  • Detecting hotspots or slow-running application components

Implementations should ensure thread-safe access to internal performance-tracking structures.
