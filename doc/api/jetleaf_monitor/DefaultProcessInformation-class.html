DefaultProcessInformation

DefaultProcessInformation class final

Default implementation of ProcessInformation that provides metadata about the current running Dart process and the underlying operating system.

This class surfaces runtime characteristics such as memory usage, processor count, OS information, locale, and host details. It also exposes custom inputs such as the tracked resident memory values (_currentRss, _freedMemory, _maxRss) and an explicit version identifier supplied during instantiation.

JetLeaf components use this implementation when collecting environment diagnostics, telemetry, or performance insights.

Memory Values

  • _currentRss: Current Resident Set Size (RSS) of the process
  • _freedMemory: Amount of memory freed since last measurement
  • _maxRss: Maximum RSS observed during the process lifetime

Version Field

  • _version: A JetLeaf-defined or user-defined version string

Notes

Platform-derived values (locale, host name, processors, OS, etc.) are resolved through the native Dart Platform API.
